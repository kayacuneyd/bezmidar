# DijitalMentor

Gurbet çocuklarına kendi dilinden öğretmen.

## Proje Hakkında
DijitalMentor, Almanya'daki Türk öğrencileri, Türkçe konuşan öğretmenlerle buluşturan bir eğitim platformudur.

## Geliştirme
Bu proje SvelteKit ve PHP (API) kullanılarak geliştirilmiştir.

### Kurulum
\`\`\`bash
npm install
npm run dev
\`\`\`

### Deployment
\`\`\`bash
./deploy.sh
\`\`\`
